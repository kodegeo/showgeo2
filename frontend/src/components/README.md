# Shared Components

Reusable UI components used across features.

## Components

### EntityContextSwitch
Allows users to switch between their user account and entity accounts.

### AssetUpload
File upload component with drag-and-drop support, preview, and type selection.

### AssetGallery
Displays a grid of assets with preview and delete functionality.

### AssetPreview
Full-screen asset preview modal with download and delete options.

### ProtectedRoute
Route wrapper component that protects routes requiring authentication. Supports role-based access control.

### Navigation
Main navigation bar with logo, links, authentication state, and entity context switch.

### Footer
Site footer with links, brand information, and copyright notice.
